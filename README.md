# eventos-database-Cabana-Multiversal
Este é um projeto feito pro evento "Imersão Dev Alura com Google Gemini" de Novembro de 2025! Um site que checa dados de eventos passados (e ainda em trabalho) da comunidade Cabana Multiversal! :D
Mas **deixo claro que boa parte do código é template da Alura e pedidos feitos em aula pro Google Gemini, incluindo vários comentários adicionados no código**
